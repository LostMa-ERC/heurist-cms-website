Repositories and cities

If you want to add a new repository, you might also need to add a new city (Place).

Place

  • place_name: The name by which the location is most commonly known.
  • administrative_region: Equivalent to GeoNames' administrative region 1, the name of the state, province, or d√©partment in which a city, town, or municipality is located.
  • country: From the Country controlled vocabulary, the country in which the place is located.
  • place_type: The type or category that describes the place.
    • City
    • Region
    • State
  • location_mappable: The string representation of a geospatial point or polygon.
    1. Find the place in GeoNames' database: https://www.geonames.org/advanced-search.html?.
    2. Copy the place's longitude (second number on GeoNames' site).
    3. Copy the place's latitude (first numebr on GeoNames' site).
    4. If entering manually:
      1. Click on the data field, then click on the "Add Geometry" button the right-hand side of the pop-up window.
      2. Past the longitude, then a comma and a space, then the latitude.
      3. Click Yes. Then click Save.
    5. If entering via CSV import:
      1. In the column, compose the following string of characters: "POINT", a space, the longitude, a comma, a space, the latitude.
  • location_certainty: The confidence with which the location of the place is ascribed.
    1. High confidence
    2. Probable location
    3. Conjectual
    4. Uncertain
    5. Mobile
  • geonames_id: The ID of the place in the GeoNames database.
    1. Find the place in GeoNames' database: https://www.geonames.org/advanced-search.html?.
    2. Copy the place's ID number from the GeoNames website.

Repository

  • preferred_name: The name by which the archival organization is most commonly known.
  • city: The city, town, or municipality in which the organization is located.
    • You may need to create a Place if one is not already in the database.
  • alternative_name: Relatively less common names by which the organization is known.
  • VIAF: The ID number in the Virtual Identity Authority File (VIAF) database that corresponds to the same organization.
    1. Look for the organization in VIAF's database: https://viaf.org/.
    2. Select the entry that is identical with the organization, i.e. https://viaf.org/viaf/51167864093422741819/#Biblioteca_Riccardiana_e_Moreniana.
    3. Copy and enter the organization's VIAF ID number, i.e. "51167864093422741819"
  • biblissima_identifier: The ID number in the Biblissima+ database that corresponds to the organization.
    1. Look for the organization in the Biblissima+ database: https://data.biblissima.fr/w/Sp%C3%A9cial:Recherche.
    2. Select the entry that is identical with the organization, i.e. https://data.biblissima.fr/w/Item:Q79184.
    3. Copy and enter the organization's Q-Entity number, i.e. "Q79184"
  • website: The homepage of the organization.

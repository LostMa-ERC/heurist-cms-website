To make edits in the database, you must be logged in. You can log in and/or access the database from this website.

  1. If your login is not already saved in your browser's cookies, you'll see a login button in the top right-hand corner of the screen. Click it.
  2. From the pop-up window, log into the database.
  3. Once logged in, you'll see a link appear in the left-hand corner of the screen on the word "database." Clicking that will open a new tab for the database.

If you have permissions to edit the website, you will also see a link further left at the top of the screen on the word "website editor." Do not click this.

Populate page

To enter new records in the database, click on the teal-colored "Populate" tab. This will reveal two options.

  • Manually enter a new record, using the website's interface and taking advantage of immediate data type verification.
  • Upload multiple records via a CSV file.

Creating records manually

From the "Populate" page, hover your mouse over the first row and select the type of record you want to create.

Once you select the record type, a window will pop up whose tabs/sections will present the fields in which you can enter information. Required fields are displayed in red, and an error message will prevent you from saving the record if you have not filled in all the required fields.

Creating records via CSV import

From the "Populate" page, click on "Delimited text/CSV."

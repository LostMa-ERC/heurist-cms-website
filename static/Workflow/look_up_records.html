Looking up records

Looking up records is useful in the following tasks:

  • Checking if something is already in the database.
  • Reviewing all records based on a criteria (i.e. the review status is "Needs action").
  • Editing a record.
  • Exporting a set of records as a CSV file.

Step 1.

Click on the Filter Builder button.

Step 2.

Select the type of record you want to explore.

Step 3.a

Select one of the record type's data fields that you want to use as the basis of your search.

Step 3.b

For data fields that link to other records (i.e. is_preserved_in on the Document record type) you can go a step further and create a filter based on data fields of the linked record type (i.e. the city of a Repository).

Step 4.a

Specify the condition you want the data field to satsify.

Step 4.b

If the data field is a linked record, you can select a specific record for the filter.

Step 5.

Once your condition or conditions are set, click the Filter button on the bottom right of the pop-up window.

Step 6.

The filter will be translated into Heurist's own language (i.e. {"t": "106"}), which we can see in the box at the top of the page.

The records that satisfy the filter's condition (i.e. manuscripts located in Paris) are listed below the filter.

You can view an individual record by clicking on it in the list and selecting the tab "Record" from the right pane.

To export all the records that satisfy the filter's condition, select the "Export" tab, click the "CSV" button, and proceed to select the data fields you want included in your CSV export.

Editing existing records

Once you find the record you're looking for, you can edit it.

  1. Click on the record from the list of search results.
  2. Make sure the "Record" tab on the right-hand pane on the Explore page is selected.
  3. Click on the pencil next to the record's Heurist ID number. This will open a pop-up window that is identical to the window used to create a new record. However, the existing record's information will pre-populate the data fields.
  4. Edit the record's data fields.
  5. Either save your changes or drop the changes, and close the pop-up window using the "Close" button.

Definition

Formulation of a Work in human language, whose literary form and style can be detected and whose creation can be attributed to one or more individuals.

Relationships

Relationships that are registered on the Text entity itself / originating from it.

  • Required
    • Text must relate to 1 Work.
    • Text must relate to 1 Genre.
    • Text must be related to 1 term from Language.
  • Optional
    • Text can relate to 0, 1, or more Hypothesis .
    • Text can relate to 0, 1, or more Text.
    • Text can relate to 0, 1, or more Person.
    • Text can relate to 0 or 1 Place.
    • Text can relate to 0, 1, or more Citation.
Example of Hypothetical Version

The oldest Witness to the chanson de Roland is in the Bodleian library's Digby 23 manuscript, written between 1129 and 1165. Scholars attest that the Anglo-Norman Text manifest in Digby 23 is based on an older, now-lost Text written between 1040 and 1115, which probably did not include some of the episodes present in Digby 23, such as the latter's Baligant section.

When creating the Text entity, set "is hypothetical" to "True" and create a Hypothesis entity, shown below, that links to the lost Franco-Norman text and details the claim attesting to its existence. The hypothetical Text can be related to more than one Hypothesis, which has its own degree of certainty and draws on its own sources of information to make the argument.

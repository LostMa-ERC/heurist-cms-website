Definition

An physical document, either currently conserved in an archival institution or attested to have previously existed as a codicological unit.

Relationships

Relationships that are registered on the Document entity itself / originating from it.

  • Required
  • Optional
    • Document can relate to 0 or 1 Repository.
    • Document can have 0, 1, or many Hypothesis.
    • Document can have 0, 1, or many Digitization.
    • Document can have 0, 1, or many Citation.
Example of Hypothetical Version

Fragments from the Anglo-Norman texts of the chanson d'Otinel (M) and chanson d'Aspremont (P4) are currently conserved in a collection of fragments at the Bibliothèque nationale de France, under the shelfmark NAF 5094. There is another fragmentary witness (C) of the same Anglo-Norman chanson d'Aspremont text in a different document, I F2, in the Clermont Ferrand archives départementales. Scholars have argued that the two fragments of chanson d'Aspremont (P4, C) were originally created as one, representing different parts of the same text. Furthermore, the fragment of chanson d'Otinel (M), which was discovered and conserved with that of chanson d'Aspremont in BNF NAF 5094, was also originally part of the same manuscript that had the complete chanson d'Aspremont.

To recognize that these witnesses (M, P4, C) were originally transmitted in the same document, create a Document entity and set the attribute "is hypothetical" to "True." Then, link the Witness entities to the hypothetical Document. They will then each be linked to two Document entities, one real, by which they can be consulted, and the other attested, which supports an historical hypothesis. Lastly, to document the argument or arguments attesting to the hypothetical manuscript's existence, create a Hypothesis entity and link it to the Document that has "is hypothetical" set to "False."
